MetroVault สามารถเปลี่ยนเครื่อง Android เครื่องเก่าให้กลายเป็นอุปกรณ์ลงนาม Bitcoin แบบออฟไลน์เฉพาะสำหรับ Bitcoin Safe ได้ อย่าติดตั้งบนโทรศัพท์ที่คุณใช้ประจำทุกวัน รีเซ็ตอุปกรณ์ก่อน เก็บมันไว้ในสถานะ **ออฟไลน์** และตัดขาดจากอินเทอร์เน็ต และใช้มันเพื่อการลงนามเท่านั้น

## สิ่งที่ควรคาดหวัง

- รีเซ็ตอุปกรณ์ Android เครื่องเก่าก่อนติดตั้ง และใช้มันเป็นอุปกรณ์ลงนามเท่านั้น
- อย่าติดตั้งแอปอื่น อย่าเข้าสู่ระบบบัญชีใด ๆ อย่าใส่ซิมการ์ด และอย่าเชื่อมต่ออุปกรณ์กลับสู่อินเทอร์เน็ตอีก
- แลกเปลี่ยนข้อมูลกระเป๋าเงิน descriptor และ PSBT กับ Bitcoin Safe ผ่าน QR code
- ลงทะเบียน descriptor แบบ multisig และส่งออกข้อมูลอุปกรณ์ลงนามสำหรับ Bitcoin Safe
- ตรวจสอบรายละเอียดทั้งหมดบนหน้าจอ MetroVault ก่อนลงนาม

## รีเซ็ตอุปกรณ์ Android และใช้มันเป็นอุปกรณ์ลงนามออฟไลน์เท่านั้น

เอกสารต้นทางของ MetroVault มี [การตั้งค่าอุปกรณ์ที่แนะนำ](https://github.com/gorunjinian/MetroVault/#recommended-device-setup) ให้ถือว่านี่เป็นข้อบังคับ: ใช้โทรศัพท์เฉพาะทางที่รีเซ็ตเป็นค่าโรงงานแล้ว อยู่ในสถานะ **ออฟไลน์** หลังการตั้งค่า และใช้เพื่อการลงนามเท่านั้น

1. รีเซ็ตโทรศัพท์เป็นค่าโรงงานก่อนติดตั้ง
2. ข้ามการตั้งค่าบัญชี Google และเก็บอุปกรณ์ไว้แบบออฟไลน์
3. เปิดโหมดเครื่องบินไว้ตลอดเวลา
4. ปิด Wi-Fi, Bluetooth, NFC และข้อมูลมือถือ
5. ถอดซิมการ์ดออก
6. ปิดการดีบัก USB
7. อย่าติดตั้งแอปอื่น และอย่าใช้โทรศัพท์ทำอย่างอื่นนอกจากการลงนาม

สำหรับการใช้งานจริง ให้ build จาก source code เมื่อทำได้ หากคุณไม่สามารถทำให้อุปกรณ์คงสภาพออฟไลน์และใช้เพื่อการลงนามเท่านั้นได้ ก็อย่าใช้กับเงินทุนจริงของ MetroVault
