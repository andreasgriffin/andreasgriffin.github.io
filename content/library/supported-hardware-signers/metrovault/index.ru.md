MetroVault может превратить старое Android-устройство в выделенное офлайн-устройство для подписи Bitcoin для Bitcoin Safe. Не устанавливайте его на свой повседневный телефон. Сначала сбросьте устройство, держите его **ОФЛАЙН** и отключенным от интернета, и используйте только для подписи.

## Чего ожидать

- Сбросьте старое Android-устройство перед установкой и используйте его только как устройство для подписи.
- Не устанавливайте другие приложения, не входите ни в какие аккаунты, не вставляйте SIM-карту и не подключайте устройство обратно к интернету.
- Обменивайтесь данными кошелька, дескрипторами и PSBT с Bitcoin Safe с помощью QR-кодов.
- Регистрируйте multisig-дескрипторы и экспортируйте данные подписывающего устройства для Bitcoin Safe.
- Проверяйте все детали на экране MetroVault перед подписью.

## Сбросьте Android-устройство и используйте его только как офлайн-подписывающее устройство

В исходной документации MetroVault теперь есть подробное [руководство по настройке устройства](https://github.com/gorunjinian/MetroVault/blob/main/docs/DEVICE_SETUP.md). Мы настоятельно рекомендуем следовать ему шаг за шагом и считать это обязательным требованием: используйте выделенный телефон после сброса к заводским настройкам, **ОФЛАЙН**, который после настройки остается отключенным от интернета и используется только для подписи.

1. Сбросьте телефон к заводским настройкам перед установкой.
2. Пропустите настройку аккаунта Google и держите устройство офлайн.
3. Оставьте авиарежим включенным постоянно.
4. Отключите Wi-Fi, Bluetooth, NFC и мобильные данные.
5. Извлеките SIM-карту.
6. Отключите отладку по USB.
7. Не устанавливайте другие приложения и не используйте телефон ни для чего, кроме подписи.

Для использования в продакшене собирайте из исходного кода, когда это возможно. Если вы не можете держать устройство офлайн и использовать его только для подписи, не применяйте его для продакшен-средств MetroVault.
