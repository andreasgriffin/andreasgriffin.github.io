    2a. Uso del Sitio Web

            - Intentar obtener acceso no autorizado a cualquier parte del Sitio Web o a sus sistemas relacionados.
            - Participar en cualquier actividad que interrumpa o interfiera con la funcionalidad del Sitio Web.
            - Usar el Sitio Web para fines ilícitos.

    2b. Uso del Software

        - Uso permitido: Puedes usar el software Bitcoin Safe de acuerdo con su licencia de código abierto, que es la [GPL3](https://www.gnu.org/licenses/gpl-3.0.en.html).

            - Usar el Software en violación de cualquier ley o normativa aplicable.
            - Modificar o distribuir el Software de una manera no permitida por su licencia.
            - Usar el Software para infringir los derechos de otros.
