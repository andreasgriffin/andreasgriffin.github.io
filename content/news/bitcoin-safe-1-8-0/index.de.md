## Neue Funktionen

#### Zoom fuer das Guthaben-Diagramm

Es wurde ein Zoom fuer das Guthaben-Diagramm hinzugefuegt ([#492](https://github.com/andreasgriffin/bitcoin-safe/pull/492)).

{{< video src="zoom.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 Zoom fuer das Guthaben-Diagramm" description="Vorschau auf den Zoom fuer das Guthaben-Diagramm in Bitcoin Safe 1.8.0." >}}

#### Benutzerdefiniertes nLocktime

Es wurde Unterstuetzung fuer benutzerdefiniertes nLocktime hinzugefuegt. Damit kann festgelegt werden, ab welcher Blockhoehe oder Uhrzeit eine Transaktion bestaetigt werden darf ([#481](https://github.com/andreasgriffin/bitcoin-safe/pull/481)).

{{< video src="nLocktime.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 benutzerdefiniertes nLocktime" description="Vorschau auf benutzerdefiniertes nLocktime in Bitcoin Safe 1.8.0." >}}

#### Bluetooth-Unterstuetzung fuer Jade

Es wurde Bluetooth-Unterstuetzung fuer Jade hinzugefuegt ([#517](https://github.com/andreasgriffin/bitcoin-safe/pull/517)).

{{< video src="bluetooth.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 Bluetooth-Unterstuetzung fuer Jade" description="Vorschau auf die Bluetooth-Unterstuetzung fuer Jade in Bitcoin Safe 1.8.0." >}}

#### App-Sperrmodus

Es wurde ein App-Sperrmodus hinzugefuegt, mit dem die App gesperrt werden kann, waehrend sie im Hintergrund weiterlaeuft ([#519](https://github.com/andreasgriffin/bitcoin-safe/pull/519)).

{{< video src="app-lock.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 App-Sperrmodus" description="Vorschau auf den App-Sperrmodus in Bitcoin Safe 1.8.0." >}}

Enthalten ist ausserdem eine nahtlose Update-Funktion fuer Windows sowie einfachere Updates unter macOS und Linux ([#513](https://github.com/andreasgriffin/bitcoin-safe/pull/513)).

## Verbesserungen

- Der Benachrichtigungsverlauf ist jetzt ueber das Tray-Symbol verfuegbar ([#493](https://github.com/andreasgriffin/bitcoin-safe/pull/493))
- Fehlerbehebungen und kleine UI-Verbesserungen
