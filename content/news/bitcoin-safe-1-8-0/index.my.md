## လုပ္ေဆာင္ခ်က္အသစ္မ်ား

#### လက္က်န္ဇယား zoom

လက္က်န္ဇယားအတြက္ zoom ကို ထည့္သြင္းထားပါသည္ ([#492](https://github.com/andreasgriffin/bitcoin-safe/pull/492)).

#### custom nLocktime

custom nLocktime အတြက္ ပံ့ပိုးမႈ ထည့္သြင္းထားၿပီး၊ transaction တစ္ခုကို အတည္ျပဳႏိုင္မည့္ အစောဆံုး block height သို႔မဟုတ္ အခ်ိန္ကို သတ္မွတ္ႏိုင္ပါသည္ ([#481](https://github.com/andreasgriffin/bitcoin-safe/pull/481)).

{{< video src="nLocktime.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 custom nLocktime" description="Bitcoin Safe 1.8.0 ရွိ custom nLocktime ပံ့ပိုးမႈ အၾကိဳၾကည့္။" >}}

#### Jade အတြက္ Bluetooth ပံ့ပိုးမႈ

Jade အတြက္ Bluetooth ပံ့ပိုးမႈ ထည့္သြင္းထားပါသည္ ([#517](https://github.com/andreasgriffin/bitcoin-safe/pull/517)).

{{< video src="bluetooth.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 Jade အတြက္ Bluetooth ပံ့ပိုးမႈ" description="Bitcoin Safe 1.8.0 ရွိ Jade Bluetooth ပံ့ပိုးမႈ အၾကိဳၾကည့္။" >}}

#### app lock mode

app lock mode ကို ထည့္သြင္းထားၿပီး၊ app သည္ ေနာက္ခံတြင္ ဆက္လက္လည္ပတ္ေနစဥ္ lock လုပ္ႏိုင္ေသာ နည္းလမ္းတစ္ခု ျဖစ္ပါသည္ ([#519](https://github.com/andreasgriffin/bitcoin-safe/pull/519)).

{{< video src="app-lock.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 app lock mode" description="Bitcoin Safe 1.8.0 ရွိ app lock mode အၾကိဳၾကည့္။" >}}

Windows အတြက္ seamless update functionality ႏွင့္ macOS၊ Linux အတြက္ ပိုမိုလြယ္ကူေသာ updates မ်ားလည္း ပါဝင္ပါသည္ ([#513](https://github.com/andreasgriffin/bitcoin-safe/pull/513)).
