## Fitur baru

#### Zoom grafik saldo

Zoom untuk grafik saldo telah ditambahkan ([#492](https://github.com/andreasgriffin/bitcoin-safe/pull/492)).

{{< video src="zoom.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 zoom grafik saldo" description="Pratinjau zoom grafik saldo di Bitcoin Safe 1.8.0." >}}

#### nLocktime kustom

Dukungan untuk nLocktime kustom telah ditambahkan, sehingga Anda dapat menentukan tinggi blok atau waktu paling awal saat transaksi bisa dikonfirmasi ([#481](https://github.com/andreasgriffin/bitcoin-safe/pull/481)).

{{< video src="nLocktime.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 nLocktime kustom" description="Pratinjau dukungan nLocktime kustom di Bitcoin Safe 1.8.0." >}}

#### Dukungan Bluetooth untuk Jade

Dukungan Bluetooth untuk Jade telah ditambahkan ([#517](https://github.com/andreasgriffin/bitcoin-safe/pull/517)).

{{< video src="bluetooth.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 dukungan Bluetooth untuk Jade" description="Pratinjau dukungan Bluetooth untuk Jade di Bitcoin Safe 1.8.0." >}}

#### Mode kunci aplikasi

Mode kunci aplikasi telah ditambahkan, yaitu cara untuk mengunci aplikasi saat tetap berjalan di latar belakang ([#519](https://github.com/andreasgriffin/bitcoin-safe/pull/519)).

{{< video src="app-lock.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 mode kunci aplikasi" description="Pratinjau mode kunci aplikasi di Bitcoin Safe 1.8.0." >}}

Juga disertakan fungsi pembaruan mulus di Windows serta pembaruan yang lebih mudah di macOS dan Linux ([#513](https://github.com/andreasgriffin/bitcoin-safe/pull/513)).
