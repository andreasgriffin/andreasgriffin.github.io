## 新機能

#### 残高チャートのズーム

残高チャートのズームを追加しました ([#492](https://github.com/andreasgriffin/bitcoin-safe/pull/492))。

{{< video src="zoom.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 残高チャートのズーム" description="Bitcoin Safe 1.8.0 の残高チャートズームのプレビュー。" >}}

#### カスタム nLocktime

カスタム nLocktime のサポートを追加しました。これにより、取引が確認可能になる最初のブロック高または時刻を設定できます ([#481](https://github.com/andreasgriffin/bitcoin-safe/pull/481))。

{{< video src="nLocktime.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 カスタム nLocktime" description="Bitcoin Safe 1.8.0 のカスタム nLocktime のプレビュー。" >}}

#### Jade の Bluetooth 対応

Jade の Bluetooth 対応を追加しました ([#517](https://github.com/andreasgriffin/bitcoin-safe/pull/517))。

{{< video src="bluetooth.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 Jade の Bluetooth 対応" description="Bitcoin Safe 1.8.0 の Jade Bluetooth 対応のプレビュー。" >}}

#### アプリロックモード

アプリロックモードを追加しました。これは、アプリがバックグラウンドで動作したままロックする方法です ([#519](https://github.com/andreasgriffin/bitcoin-safe/pull/519))。

{{< video src="app-lock.mp4" controls=true muted=true playsinline=true loop=true uploadDate="2026-03-31" name="Bitcoin Safe 1.8.0 アプリロックモード" description="Bitcoin Safe 1.8.0 のアプリロックモードのプレビュー。" >}}

Windows でのシームレスな更新機能と、macOS および Linux でより簡単な更新も含まれています ([#513](https://github.com/andreasgriffin/bitcoin-safe/pull/513))。

## 改善点

- 通知履歴がトレイアイコンから利用可能になりました ([#493](https://github.com/andreasgriffin/bitcoin-safe/pull/493))
- バグ修正と小さな UI 改善
